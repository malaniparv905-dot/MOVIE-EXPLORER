# Movie Explorer Web App

## 📌 Project Description
Movie Explorer is a web application that allows users to search for movies and view detailed information such as title, poster, release year, rating, genre, and plot.

The purpose of this project is to practice working with APIs, JavaScript, and web development concepts such as search, filtering, and dynamic content display.

---

## 🌐 API Used
This project uses the OMDb API (Open Movie Database API).

API Link:
https://www.omdbapi.com/

The API provides movie data including:
- Movie title
- Poster image
- Release year
- Genre
- IMDB rating
- Plot summary

---

## ✨ Features
Planned features of the application:

1. Search movies by name
2. Display movie details
3. Show movie poster
4. Filter movies by year
5. Sort movies by rating
6. Responsive user interface
7. Save favorite movies (optional feature)
